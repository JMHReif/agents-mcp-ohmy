= Agents, Tools, and MCP, oh my!

This repository showcases next-level concepts (beyond vectors and RAG) for developers working on AI projects.

== Running the application

Set environment variable (if applicable) for Large Language Model - e.g. OpenAI.

[source,shell]
----
source ./env.sh
----

Run the application.

[source,shell]
----
./mvnw spring-boot:run
----

Test the endpoints.

[source,bash]
----
#Traditional systems
http ":8080/act1/five-star-books"
http ":8080/act1/books-not-read?userId=8842281e1d1347389f2ab93d60773d4d"
http ":8080/act1/count-read-books?userId=8842281e1d1347389f2ab93d60773d4d"

#Agentic tools
##Count books method
http ":8080/act2/stage1/query?userId=8842281e1d1347389f2ab93d60773d4d&userQuery=how many books have I read?"
##Find books not read method
http ":8080/act2/stage1/query?userId=8842281e1d1347389f2ab93d60773d4d&userQuery=could you find well-rated books for me to read?"
##Best rated books method
http ":8080/act2/stage1/query?userId=8842281e1d1347389f2ab93d60773d4d&userQuery=what are the best rated books?"
##RAG method (reviews)
http ":8080/act2/stage1/query?userQuery=could you find book reviews where readers changed their perspective throughout the book?"
##GraphRAG
http ":8080/act2/stage1/query?userId=8842281e1d1347389f2ab93d60773d4d&userQuery=I want to find fantasy books about dragons and magic"
http ":8080/act2/stage1/query?userId=8842281e1d1347389f2ab93d60773d4d&userQuery=what are some nonfiction books on productivity?"
